
# To Subs

Convert your sheet files to a subtitle file (.ass).

## Sheet Example

| Time | Original | Translation |
| ---- | -------- | ----------- |
| 0:00 | Hello    | 你好        |
| 1:00 | World    | 世界        |

## Usage

```bash
# npm user
npx to-subs <SHEET_FILE_PATH>

# pnpm user
pnpx to-subs <SHEET_FILE_PATH>
```

## Supported Ext

- xlsx
- csv
